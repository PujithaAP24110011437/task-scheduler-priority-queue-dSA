
📌 Task Scheduler using Priority Queue (Heap)

This project implements an efficient Task Scheduling System using the Priority Queue (Heap) data structure, a key concept in Data Structures and Algorithms (DSA). The scheduler processes tasks based on their priority, ensuring that higher-priority tasks are executed before lower-priority ones.

🚀 Features
📊 Priority-based task execution
⚡ Efficient operations using Min Heap / Max Heap
➕ Add new tasks with priority
❌ Remove/execute highest priority task
🔄 Dynamic task management
🧠 Optimized time complexity (O(log n) for insertion & deletion)

▶️ How It Works

Tasks are inserted into a priority queue where each task is assigned a priority value. The scheduler always selects the task with the highest priority (or lowest value in case of a min-heap) for execution, ensuring efficient and optimized scheduling.
